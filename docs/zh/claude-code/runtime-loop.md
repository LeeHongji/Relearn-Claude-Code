# 运行时主循环

Claude Code 的主循环是它从“CLI 包壳”变成“真正 agent runtime”的关键。

## 一条最重要的调用脊柱

`main.tsx → QueryEngine.ts → query.ts → StreamingToolExecutor / toolOrchestration`

## 各阶段职责

### 阶段 1：准备当前 turn
`QueryEngine.ts` 负责会话级状态和上下文拼装。

### 阶段 2：调用模型
`query.ts` 发送消息并接收流式输出。

### 阶段 3：流式渲染
系统不会等整段结果结束再统一展示。

### 阶段 4：执行工具
工具可能并发执行，也可能串行执行，取决于是否安全。

### 阶段 5：把结果写回上下文
工具结果会回流进消息历史和上下文状态。

### 阶段 6：决定继续 / 重试 / 压缩 / 停止
这一步决定了系统是否具备“韧性”。

## 一个关键工程结论

主循环不是细节，它就是整个 agent 产品的“操作系统”。
